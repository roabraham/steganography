1 # Simple Image Steganography
2 
3 This web application provides a simple way for effective image _steganography_ designed to hide larger amount of data in an image file by replacing an entire _color component_ of the _carrier image_ instead of a few bits so you can hide much more data than a simple message. On the con, this method is much weaker than the traditional image steganorgaphy so you should choose a carrier image with a strong _dominant color palette_ and hide the data by replacing the _weakest color component_ to make this steganography less noticable. _If it is essential to hide your data properly, you should choose a different steganography tool_ as this module was designed to hide large amount of data instead of hiding data deep in the carrier image to make finding the data nearly impossible.

17 ## Embedding PHP steganography class in a different web application
18 
19 To _embed PHP steganography class_ in a separate web application, place `class.php_stego.php` file in your web application and include the class in the corresponding script as you can see bellow:
20 
21 ```php
22 require_once 'class.php_stego.php';
23 ...
24 $php_stego = new PHP_STEGO();
25 //if $bin_to_image = 1, binary data fill be converted to an image
26 //binary data will be extracted from the image otherwise
27 $php_stego->set_encoding_direction($bin_to_image);
28 $php_stego->set_carrier_data($carrier_image_data);
29 $php_stego->set_input_data($binary_data_to_encode);
30 $php_stego->set_encryption_key('Your Password');
31 $output_image_data = $php_stego->convert();
32 ```
33 
34 To extract the encoded binary data, do the following:
35 
36 ```php
37 ...
38 $php_stego->set_input_data($output_image_data);
39 $php_stego->set_encoding_direction(false);
40 $output_file = $php_stego->convert();
41 ```
